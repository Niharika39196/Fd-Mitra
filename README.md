# FD Mitra

FD Mitra is a multi-language voice-capable Fixed Deposit advisory application.

## Prerequisites
- Node.js 18+
- Python 3.10+

## Backend Setup

1. Navigate to the `backend` directory: `cd backend`
2. Install Python dependencies: `pip install -r requirements.txt`
3. Configure environment variables in `.env`
4. Start the backend server: `uvicorn main:app --reload`

The backend will run on `http://localhost:8000`.

## Frontend Setup

1. Navigate to the `frontend` directory: `cd frontend`
2. Install Node dependencies: `npm install`
3. Start the Vite development server: `npm run dev`

The frontend will run on `http://localhost:5173`. Open this URL in your browser.
